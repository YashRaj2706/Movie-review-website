# Movie-review-website
# 🎬 Movie Review site (Flask + SQLAlchemy)

A simple Movie Review Website  i.e.built using "Flask" and "SQLite" database.  
Users can add movies, write reviews, update reviews, delete reviews, and view average ratings.

---

# 🚀 Features

- ✅ Add New Movie
- ✅ Search Movies
- ✅ Add Review for a Movie
- ✅ Update Review
- ✅ Delete Review
- ✅ Display Average Rating
- ✅ One-to-Many Relationship (Movie → Reviews)

---

# 🛠 Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML (Jinja2 Templates)
